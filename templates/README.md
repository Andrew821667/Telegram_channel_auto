# Шаблоны изображений для публикаций

Эта директория содержит базовое изображение-шаблон для всех публикаций в Telegram канале.

## Быстрый старт

### Автоматическая генерация шаблона

Запустите скрипт для создания базового градиентного фона:

```bash
python3 create_template.py
```

Или внутри Docker контейнера:

```bash
docker exec -it legal_ai_app python3 create_template.py
```

### Использование своего изображения

1. Подготовьте изображение размером **1200x630 пикселей** (стандарт для Telegram)
2. Сохраните его как `base_template.jpg` в эту директорию
3. Убедитесь, что изображение не слишком пестрое - на него будет накладываться белый текст

## Рекомендации по дизайну

### Размеры
- **Ширина:** 1200 px
- **Высота:** 630 px
- **Формат:** JPG или PNG
- **Качество:** Высокое (>90%)

### Цвета и контраст
- Используйте темные или средние тона (на них накладывается белый текст с тенью)
- Избегайте слишком ярких или белых участков в центре
- Оптимально: градиенты, приглушенные фоны, корпоративные цвета

### Безопасная зона
- **Центр изображения:** Сюда накладывается заголовок (оставьте свободным от важных деталей)
- **Правый верхний угол:** Дата публикации
- **Правый нижний угол:** Водяной знак "AI Legal News"

## Примеры использования

### Вариант 1: Корпоративный стиль
Используйте изображение с вашим логотипом в углу и фирменным градиентом

### Вариант 2: Тематический фон
Абстрактные изображения на тему AI, права, технологий (с темным оверлеем)

### Вариант 3: Минимализм
Простой градиент или однотонный фон (создается автоматически скриптом)

## Настройка в .env

Путь к шаблону настраивается через переменную окружения:

```env
MEDIA_TEMPLATE_IMAGE_PATH=/app/templates/base_template.jpg
```

Вы можете создать несколько шаблонов и переключаться между ними, меняя эту настройку.

## Fallback

Если файл шаблона не найден, система автоматически создаст градиентный фон на лету.
